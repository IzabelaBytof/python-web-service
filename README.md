## Pierwsze uruchomienie

Linux/macos

```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

Windows

```
py -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
```

## Start aplikacji

python app.py
